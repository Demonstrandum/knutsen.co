## Build with `seam`
[`seam`](https://git.knutsen.co/seam)
(**s**ymbolic **e**xpressions **a**s **m**arkup)
is needed to build the `.sex` (**s**ymbolic **ex**pression) files.

After that, it's just to run the Makefile:
```sh
$ make clean
$ make
```
